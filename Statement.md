## Problem Statement
To design and develop a console-based ATM simulation system that allows users to authenticate themselves and perform basic banking operations without the need for a database or GUI.

## Scope of the Project
- The system will support multiple pre-defined accounts.
- Users can log in using account number and PIN.
- Users can perform ATM functions such as withdrawing, depositing, transferring funds, and checking balance.
- Includes security features like PIN verification and attempt limit.
- Mini-statement functionality shows recent transactions.

## Target Users
- Students learning Java  
- Beginners in software development  
- Anyone wanting to understand how ATM logic works  

## High-Level Features
1. Login & Authentication  
2. Check Balance  
3. Deposit Money  
4. Withdraw Money  
5. Transfer Money  
6. Mini Statement  
7. Change PIN  
8. Logout/Exit the ATM
